# Privacy Policy — Compass Modernizer

Compass Modernizer does not collect, store, or transmit any personal or sensitive user data.

The extension only modifies the visual appearance of the Compass Education website by applying themes and UI styling. All processing happens locally in the user’s browser.

The extension does not access page content, user activity, authentication information, or any other data.

The only data stored is the user’s theme preference (dark/light mode and custom colors), which is saved locally on the device and never sent anywhere.

The extension does not include analytics, tracking scripts, advertising, or third‑party code.

If the user clicks the optional “Send Feedback” button, a Microsoft Forms page opens in a new tab. No information is transmitted by the extension itself.
